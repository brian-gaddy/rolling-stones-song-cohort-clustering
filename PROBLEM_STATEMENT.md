# Problem Statement

Spotify wants to create cohorts of songs to improve recommendation quality and customer engagement. The project uses Spotify API data for Rolling Stones albums and songs.

The objective is to perform exploratory data analysis and cluster analysis to better understand the factors that define similar groups of songs.

## Required Work

- Inspect the dataset for duplicates, missing values, irrelevant entries, and outliers.
- Clean and refine the dataset for further processing.
- Perform exploratory data analysis and feature engineering.
- Recommend albums based on the number of popular songs.
- Analyze relationships between song popularity and song features.
- Explain the significance of dimensionality reduction techniques.
- Identify the right number of clusters.
- Apply clustering algorithms.
- Define each cluster based on its features.
