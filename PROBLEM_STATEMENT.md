# Problem Statement

Spotify-style music platforms improve engagement when they recommend content aligned with listener preferences. This project uses Spotify audio-feature data for Rolling Stones songs to create cohorts of similar songs.

## Objective

Perform exploratory data analysis and cluster analysis to better understand the features that define song similarity.

## Required Work

- Inspect the dataset for duplicates, missing values, irrelevant entries, and outliers.
- Clean and refine the dataset for further processing.
- Perform exploratory data analysis and feature engineering.
- Recommend albums based on the number of popular songs.
- Analyze relationships between song popularity and audio features.
- Explain the significance of dimensionality reduction techniques.
- Identify an appropriate number of clusters.
- Apply clustering algorithms.
- Define each cluster based on its features.
